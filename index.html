Критерии оценивания (12-балльная шкала):

Полнота проекта (4 балла):
Проект должен полностью решать поставленную задачу. Если проект не завершен или решает задачу лишь частично, это снижает баллы.

Качество кода (3 балла):
Качество и структура кода: чистота, использование функций и методов, адекватные комментарии. Если код неструктурирован, это снижает оценку.

Асинхронность и работа с API (2 балла):
Использование fetch, async/await и грамотная работа с API. Ошибки в работе с API и некорректная обработка запросов снижают оценку.

Работа с DOM (2 балла):
Интерактивность страницы: работа с DOM-элементами, обработка событий и динамическое обновление данных на странице.

Креативность и интерфейс (1 балл):
Если проект реализован с добавлением оригинальных идей или интересных интерфейсных решений, это будет плюсом.
